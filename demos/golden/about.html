Why we made this

This demo explores how a single-breed reference site could feel: calmer, more visual, and more useful than a generic brochure page. The purpose is to prove layout, navigation, search, gallery, and editorial tone before any real publication model exists.

How this demo was made

The pages use prototype copy and generated visual assets. They should not be treated as veterinary advice, breed-club guidance, or a reviewed care library.

  • Frontend structure and shared chrome are the real demo output
  • Breed and care copy needs expert review before any real launch
  • Generated imagery should be replaced or disclosed if reused
  • Search, gallery, and guide flows are included to show product shape

What this demo does not do

This demo does not sell listings, collect signups, run sponsored content, or offer paid services. Any business, legal, editorial, or privacy model would need to be designed separately before launch.

# Elusion Works Repo Instructions

## Purpose

This repo contains the public-facing `Elusion Works` brand site.

## Rules

- Keep this as a fast static site unless there is a clear reason to add a framework.
- Treat `data/estate.js` as the structured source of truth for the homepage listings.
- Keep local-only prompts, notes, and experiments in `.local/`.
- Track docs that help future setup, design, or deployment.
- Do not create duplicate scratch copies of this repo elsewhere in the estate.

## Public Repo Content

The public repo should include:

- website files
- setup/troubleshooting docs
- architecture/design docs
- brand assets intended for the live site

It should not include:

- loose handoff notes
- local planning scratchpads
- private prompts
- temporary review artefacts
